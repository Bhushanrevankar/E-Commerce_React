## Run Locally

Clone the project

```bash
  git clone https://github.com/Bhushanrevankar/React_E-Commerce-Major.git
```

Go to the project directory

```bash
  cd React_E-Commerce-Major
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
